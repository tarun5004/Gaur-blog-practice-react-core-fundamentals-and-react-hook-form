# 🧠 Naye Concepts Samjho

**`-mt-7` — Negative Margin:**
```
Avatar ko banner ke UPAR pull karta hai
Normal mt = neeche dhakelta hai
Negative -mt = upar kheenchta hai
```

**`overflow-hidden` on card:**
```
Banner bilkul card ke edges tak jaaye
Bina iske banner card se bahar nikal jaata
```

**`space-y-4` on column:**
```
Column ke andar jo bhi cards hain
unke beech automatically gap aata hai
Har item ke beech = 4 units space
```

**`text-gbtext/60`:**
```
/60 = 60% opacity
Matlab text thoda faded/muted dikhega
/50 = aur zyada faded
/70 = thoda kam faded
```
